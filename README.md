# Slack Bot Example

Use Wit AI + Slack.

## Getting Started

You need to copy `.env.example` to `.env`.

```
$ cp .env.example .env
```

The `.env` file will need your Wit.ai server access token and your Slack webhook url and verification token.

Once you have those added to your env file, you can run:

```
$ nvm use 8.0.0
$ npm i -g nodemon
$ yarn start
```
